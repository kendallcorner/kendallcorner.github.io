# Readme

Requirements: Node.js

```
git clone git@github.com:kendallcorner/kendallcorner.github.io.git

npm install
```

To develop: information at [hexo.io](https://hexo.io)

```
hexo serve
```
with automatic updates

New page:
```
hexo new post [name]
```


to deploy:
```
hexo generate

hexo deploy
```
